🚦 连接流程

手机热点方案（推荐）：
1. 手机开热点
2. 机器人连该热点
3. 电脑也连该热点
4. 查看热点设备列表找机器人IP
5. IP填入左侧，关模拟模式
6. 点「测试连接」确认

测试连接顺序：
→ ping HTTP :80 /reeman/hostname
→ 所有API均在端口 80

机器人IP一般固定在热点下
同一热点重连IP通常不变

⚡ 速度默认值
